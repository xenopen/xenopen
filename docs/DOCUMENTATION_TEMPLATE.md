### 関数一覧

| 関数名 | 説明 | 戻り値 |
|--------|------|--------|
| `functionName()` | 関数の説明 | `戻り値の型` |

### 詳細

#### `functionName(param1, param2)`

関数の詳細な説明。

**パラメータ:**

| 名前 | 型 | 必須 | デフォルト | 説明 |
|------|-----|------|------------|------|
| `param1` | `string` | ✓ | - | パラメータの説明 |
| `param2` | `number` | - | `0` | オプションパラメータの説明 |

**戻り値:**

`ReturnType` - 戻り値の説明

**例:**

```javascript
const result = functionName('hello', 42);
console.log(result); // 期待される出力
```

**例外:**

- `Error` - エラーが発生する条件
